# SreekCas
An attempt to implement a Computer Algebra using Java 

We need implement an expression tree first of all where each node is a constant variable or an operation for example 



       +
      / \
     *   3
    / \
   2   x


   evaluating this it will be 2*x + 3 

   Let's make an expression class and then add a bit of extra stuff
